# 🏥 HealthBridge

HealthBridge is a full-stack healthcare management system that streamlines interactions between patients, doctors, and administrators.

## 🚀 Tech Stack

### Backend
- Java 21
- Spring Boot 3.4.5
- Spring Security
- JWT Authentication
- MySQL
- Maven

### Frontend
- React
- Vite
- Tailwind CSS

### Features
- Secure JWT Authentication
- Role-based Access Control
- Patient Management
- Doctor Management
- Appointment Scheduling
- Medical Records
- Feedback System
- Email OTP Verification
