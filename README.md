# 🏥 HealthBridge

HealthBridge is a full-stack Healthcare Management System that simplifies communication and workflow between patients, doctors, and administrators through a secure web application.

---

# ✨ Features

## Authentication & Security

- JWT Authentication
- Email OTP Verification
- Password Reset
- BCrypt Password Encryption
- Role-Based Access Control (Admin, Doctor, Patient)

---

## Admin Module

- Dashboard
- User Management
- Department Management
- Doctor Schedule Management
- Appointment Monitoring
- Medical Record Management
- Feedback Management

---

## Doctor Module

- Doctor Dashboard
- Patient Management
- Schedule Management
- Medical Records
- Feedback Review

---

## Patient Module

- Patient Dashboard
- Book Appointment
- View Medical Records
- Submit Feedback
- Doctor Reviews
- Profile Management

---

# 🚀 Tech Stack

## Backend

- Java 21
- Spring Boot 3.4.5
- Spring Security
- Spring Data JPA
- JWT Authentication
- Spring Mail
- ModelMapper
- MySQL
- Maven
- Swagger / OpenAPI

## Frontend

- React 18
- Vite
- React Router
- Axios
- React Hook Form
- Tailwind CSS
